# Dependency Inversion — Links

Curated resources for deeper reading on DIP, ports and adapters, and dependency injection in C++.

* [Dependency Inversion Principle (Wikipedia)](https://en.wikipedia.org/wiki/Dependency_inversion_principle) — concise definition, UML diagrams, and history from Robert C. Martin's original paper.
* [SOLID Design Principles — Dependency Inversion (Arne Mertz)](https://arne-mertz.de/2020/06/solid-dependency-inversion/) — practical C++ walkthrough with before/after code.
* [Hexagonal Architecture (Ports & Adapters) Explained](https://alistair.cockburn.us/hexagonal-architecture/) — Alistair Cockburn's original description of the ports-and-adapters pattern.
* [Dependency Injection in C++ — fluentcpp](https://www.fluentcpp.com/2017/09/22/make-pimpl-using-unique_ptr/) — explores constructor injection, `unique_ptr` ownership, and the pImpl boundary.
* [CppCon: Breaking Dependencies — The SOLID Principles (Klaus Iglberger)](https://www.youtube.com/watch?v=Ntraj80qN2k) — conference talk covering DIP with real-world C++ examples.
