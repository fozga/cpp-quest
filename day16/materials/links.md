# Custom Iterators — Links

Supplementary resources for understanding iterator design and implementation
in modern C++.

- [Iterator library — cppreference](https://en.cppreference.com/w/cpp/iterator)
  Comprehensive reference for all iterator categories, traits, and adaptors.

- [Writing a custom iterator in modern C++ — Internal Pointers](https://www.internalpointers.com/post/writing-custom-iterators-modern-cpp)
  Step-by-step tutorial on building a conforming iterator from scratch.

- [std::forward_iterator_tag — cppreference](https://en.cppreference.com/w/cpp/iterator/iterator_tags)
  Details on iterator category tags and how traits dispatch on them.

- [std::accumulate — cppreference](https://en.cppreference.com/w/cpp/algorithm/accumulate)
  Reference for the numeric algorithm used in the tests.

- [Circular buffer — Wikipedia](https://en.wikipedia.org/wiki/Circular_buffer)
  Overview of ring buffer data structure, head/tail management, and use cases.

- [C++20 Iterator Concepts — cppreference](https://en.cppreference.com/w/cpp/iterator/forward_iterator)
  The concept-based formulation of forward iterator requirements in C++20.
