# Lock-Free Basics — Links

Introductory and reference material for `std::atomic` and lock-free programming.

- [cppreference — std::atomic](https://en.cppreference.com/w/cpp/atomic/atomic) — Full API reference for `std::atomic`, including `compare_exchange_weak/strong`.
- [cppreference — compare_exchange](https://en.cppreference.com/w/cpp/atomic/atomic/compare_exchange) — Detailed semantics of CAS operations, failure ordering, and spurious failures.
- [Herb Sutter — Lock-Free Programming](https://herbsutter.com/2013/02/11/atomic-weapons-the-c-memory-model-and-modern-hardware/) — "Atomic Weapons" talk covering the C++ memory model and hardware reality.
- [Preshing on Programming — An Introduction to Lock-Free Programming](https://preshing.com/20120612/an-introduction-to-lock-free-programming/) — Accessible overview of lock-free concepts with diagrams.
- [Fedor Pikus — C++ Atomics From Basic to Advanced (CppCon)](https://www.youtube.com/watch?v=ZQFzMfHIxng) — CppCon talk walking through atomic operations and common pitfalls.
