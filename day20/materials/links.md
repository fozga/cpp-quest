# Variadic Templates — Links & Resources

The following references cover variadic templates and fold expressions in depth.

- [Parameter pack — cppreference.com](https://en.cppreference.com/w/cpp/language/parameter_pack)
  Comprehensive reference for template and function parameter packs, expansion contexts, and examples.

- [Fold expressions — cppreference.com](https://en.cppreference.com/w/cpp/language/fold)
  Official reference for all four fold expression forms, supported operators, and behaviour with empty packs.

- [Variadic Templates in C++ — Eli Bendersky](https://eli.thegreenplace.net/2014/variadic-templates-in-c/)
  Accessible tutorial walking through variadic templates from basic recursion to practical examples.

- [C++17 Fold Expressions — Modernes C++](https://www.modernescpp.com/index.php/fold-expressions)
  Explains fold expressions with clear diagrams showing how each form expands.

- [Using Variadic Templates Cleanly — Fluent C++](https://www.fluentcpp.com/2021/03/12/cpp-fold-expressions/)
  Practical guide to fold expressions with real-world patterns including stream output and logical reductions.

- [CppCon: Variadic Templates — Peter Sommerlad (YouTube)](https://www.youtube.com/results?search_query=cppcon+variadic+templates+fold+expressions)
  Conference talks exploring variadic template techniques and fold expression patterns.
