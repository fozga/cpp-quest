# cpp-drills-advanced

Welcome to **cpp-drills-advanced**, a 40-day intensive, interactive learning curriculum designed for intermediate C++ developers. This course transitions you from writing basic, C-style "C with Classes" code into crafting highly optimized, modern, and robust systems-level C++. Over exactly 40 days, you will dive under the hood of the language, mastering advanced paradigms like template metaprogramming, the evolution of modern C++ standards (from C++11 up to C++26), memory management, and lock-free concurrency. Every day features a hands-on task—ranging from implementing core mechanics to debugging memory leaks and refactoring legacy code—culminating in a comprehensive 7-day final project.

## Learning Outcomes

By completing this 40-day course, you will be able to:

* **Manage Resources Flawlessly**: Apply pure RAII, the Rule of 5/0, and custom deleters, while safely utilizing memory pools to optimize smart pointer allocations.
* **Master Move Semantics**: Leverage rvalue references, perfect forwarding, and compiler optimizations (RVO/NRVO) by architecting systems that physically cannot perform unnecessary copies.
* **Demystify Advanced Polymorphism**: Understand vtable mechanics under the hood, resolve the diamond problem, and refactor dynamic polymorphism into zero-cost static polymorphism using CRTP.
* **Implement Modern Error Handling**: Completely eliminate costly exceptions from your codebase using modern C++ value types like `std::optional`, `std::variant`, and C++23 `std::expected`.
* **Write Declarative Functional C++**: Design custom iterators and replace imperative raw loops with elegant, composable C++20 Ranges and Views.
* **Harness Template Metaprogramming**: Write type traits and variadic templates, and gracefully transition from complex SFINAE constraints to highly readable C++20 Concepts.
* **Trace the Evolution of C++**: Systematically upgrade legacy codebases by applying C++11/14/17 features, C++20 coroutines, and C++23 explicit object parameters ("deducing this").
* **Build Safe Concurrent Systems**: Utilize modern threading primitives like `std::jthread`, `std::future`, and `std::promise`, while optimizing lock-free operations with relaxed memory orders.
* **Architect Complex Systems**: Design and implement an asynchronous, dependency-aware task graph execution engine from scratch.


## Theme Blocks Overview

The course is structured into carefully paced thematic blocks, typically spanning 2 to 5 days, to encourage deep, iterative learning. Instead of isolating concepts into single disconnected days, you will build upon your work daily within a block. For instance, you will first implement dynamic polymorphism, debug its memory footprint, and then systematically refactor it into highly optimized static polymorphism. By removing pure benchmarking and focusing entirely on active implementation and refactoring, you will build muscle memory for modern C++ standards, ultimately combining these advanced paradigms into the massive 7-day Async Task Graph System project.
