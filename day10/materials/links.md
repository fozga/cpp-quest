# The Diamond Problem — Links

Curated resources for deeper exploration of multiple inheritance, the diamond
problem, and virtual inheritance in C++.

- **cppreference — Derived classes (virtual base classes)**
  <https://en.cppreference.com/w/cpp/language/derived_class#Virtual_base_classes>
  Authoritative reference covering virtual inheritance syntax, initialization
  order, and conversion rules.

- **ISO C++ FAQ — Multiple and Virtual Inheritance**
  <https://isocpp.org/wiki/faq/multiple-inheritance>
  Plain-language explanations of why the diamond problem occurs and how virtual
  inheritance resolves it, with practical guidance on when to use it.

- **learncpp.com — Virtual Base Classes**
  <https://www.learncpp.com/cpp-tutorial/virtual-base-classes/>
  Step-by-step tutorial with diagrams showing duplicated vs shared subobjects.

- **Arthur O'Dwyer — "Multiple Inheritance Considered Useful"**
  <https://quuxplusone.github.io/blog/2020/12/02/multiple-inheritance/>
  A nuanced look at when multiple inheritance (including virtual) is the right
  tool, with real-world examples from the Standard Library.

- **Microsoft Learn — Virtual Base Classes**
  <https://learn.microsoft.com/en-us/cpp/cpp/virtual-base-classes>
  Compiler-vendor documentation with memory layout diagrams and MSVC-specific
  implementation notes.
