# Type Traits — Links

Supplementary resources for understanding type traits and compile-time type inspection in C++.

- [std::type_traits — cppreference](https://en.cppreference.com/w/cpp/header/type_traits) — Complete reference for the `<type_traits>` header and all standard traits.
- [std::integral_constant / true_type / false_type — cppreference](https://en.cppreference.com/w/cpp/types/integral_constant) — The building blocks that all predicate traits inherit from.
- [std::basic_string — cppreference](https://en.cppreference.com/w/cpp/string/basic_string) — Full template signature and typedefs for `std::string`, `std::wstring`, etc.
- [static_assert — cppreference](https://en.cppreference.com/w/cpp/language/static_assert) — Compile-time assertion syntax and usage since C++11/C++17.
- [Template Specialization — cppreference](https://en.cppreference.com/w/cpp/language/template_specialization) — Full and partial specialization rules for class templates.
- [An Introduction to C++ Type Traits — Bartlomiej Filipek](https://www.cppstories.com/2024/type-traits-intro/) — Practical walkthrough of writing and using custom type traits.
