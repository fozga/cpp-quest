# Custom Views — Links

Supplementary resources for understanding custom views and view adaptors in C++20.

- [std::ranges::view_interface — cppreference](https://en.cppreference.com/w/cpp/ranges/view_interface) — CRTP base for building custom view types with automatic member derivation.
- [std::ranges::filter_view — cppreference](https://en.cppreference.com/w/cpp/ranges/filter_view) — Reference for the filter view used internally by `std::views::filter`.
- [std::views::iota — cppreference](https://en.cppreference.com/w/cpp/ranges/iota_view) — Lazy sequence generator, useful for infinite integer ranges.
- [C++20 Ranges: A Complete Guide — Bartlomiej Filipek](https://www.cppstories.com/2022/ranges-composition/) — Practical guide to composing range adaptors and writing custom views.
- [Handy F. Jiang: Writing Your Own C++20 Range Adaptors](https://hannes.hauswedell.net/post/2019/11/30/range_adaptor/) — Step-by-step walkthrough of building a custom range adaptor from scratch.
