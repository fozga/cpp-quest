# Static Analysis & Code Style — Links

Curated resources on static analysis tooling and writing clean, tool-friendly C++.

- [clang-tidy Documentation (LLVM)](https://clang.llvm.org/extra/clang-tidy/)
  Official reference for all available checks, configuration, and editor integration.

- [cppcheck Manual](https://cppcheck.sourceforge.io/manual.html)
  User guide for cppcheck covering command-line usage, suppressions, and CI integration.

- [C++ Core Guidelines (isocpp)](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines)
  The authoritative style and safety guidelines that many clang-tidy checks enforce.

- [CppCon Talk — Practical C++ Static Analysis (Arne Mertz)](https://www.youtube.com/results?search_query=cppcon+practical+static+analysis+c%2B%2B)
  Conference talk walking through real-world findings from clang-tidy and cppcheck.

- [GCC Warning Options (GCC Manual)](https://gcc.gnu.org/onlinedocs/gcc/Warning-Options.html)
  Complete reference for `-Wall`, `-Wextra`, and the individual warnings they enable.

- [SonarSource C++ Rules](https://rules.sonarsource.com/cpp/)
  Browseable catalogue of all SonarQube/SonarCloud rules with rationale and examples.
