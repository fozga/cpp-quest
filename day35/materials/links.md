# Putting It All Together — Links

Curated resources on integrating testing, refactoring, and tooling into a sustainable workflow.

- [Refactoring Guru — Refactoring Techniques](https://refactoring.guru/refactoring/techniques)
  Catalogue of named refactorings with before/after examples and motivation.

- [AddressSanitizer (Google)](https://github.com/google/sanitizers/wiki/AddressSanitizer)
  Official documentation for ASan — how it works, what it catches, and how to interpret reports.

- [C++ Core Guidelines — Resource Management](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines#S-resource)
  The ownership and RAII sections of the C++ Core Guidelines.

- [CppCon Talk — Back to Basics: RAII and the Rule of Zero (Arthur O'Dwyer)](https://www.youtube.com/results?search_query=cppcon+back+to+basics+raii+rule+of+zero)
  Accessible conference talk on letting constructors and destructors manage resources.

- [Martin Fowler — Test Pyramid](https://martinfowler.com/articles/practical-test-pyramid.html)
  Practical guide to balancing unit, integration, and end-to-end tests.

- [Compiler Explorer (Godbolt)](https://godbolt.org/)
  Instantly see compiler output, warnings, and optimisation decisions for any C++ snippet.
