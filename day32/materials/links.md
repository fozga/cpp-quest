# Unit Testing Mindset — Links

Curated resources on writing effective, maintainable unit tests.

- [The Art of Unit Testing — Fundamentals (Roy Osherove)](https://www.artofunittesting.com/)
  Authoritative overview of what makes a good unit test and how to structure them.

- [Arrange-Act-Assert Pattern (C2 Wiki)](https://wiki.c2.com/?ArrangeActAssert)
  Concise explanation of the AAA pattern with rationale.

- [Unit Testing Best Practices (Microsoft Learn)](https://learn.microsoft.com/en-us/dotnet/core/testing/unit-testing-best-practices)
  Language-agnostic principles: naming conventions, isolation, single-concern tests.

- [Google Testing Blog — Testing on the Toilet](https://testing.googleblog.com/)
  Short, practical articles on test design, test smells, and maintainability.

- [Modern C++ Testing with Catch2 (GitHub)](https://github.com/catchorg/Catch2/blob/devel/docs/tutorial.md)
  Tutorial for a popular C++ test framework; useful once you move beyond raw `assert`.

- [CppCon Talk — Back to Basics: Unit Tests (Ben Saks)](https://www.youtube.com/results?search_query=cppcon+back+to+basics+unit+tests)
  Conference talk covering C++-specific testing considerations and tooling.
