# SFINAE Constraints — Links

Supplementary reading on SFINAE, `std::enable_if`, and constrained templates.

- [cppreference — SFINAE](https://en.cppreference.com/w/cpp/language/sfinae)
  Authoritative reference for the substitution-failure rule with examples.

- [cppreference — std::enable_if](https://en.cppreference.com/w/cpp/types/enable_if)
  Full specification, alias template, and usage patterns.

- [Eli Bendersky — SFINAE and enable_if](https://eli.thegreenplace.net/2014/sfinae-and-enable_if/)
  Approachable tutorial walking through SFINAE from first principles.

- [foonathan::blog — Overload Resolution and SFINAE](https://www.foonathan.net/2015/11/overload-resolution-3/)
  Deep dive into how the compiler ranks candidates and applies SFINAE.

- [Jason Turner — C++ Weekly: SFINAE](https://www.youtube.com/watch?v=mNxAqLVIaW0)
  Short video explanation with live coding examples.

- [Bartek's Coding Blog — Simplify SFINAE with C++20 Concepts](https://www.bfilipek.com/2018/11/template-constraints.html)
  Compares SFINAE patterns with the Concepts alternative.
