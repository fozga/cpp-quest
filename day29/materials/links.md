# Type Erasure — Links

Introductory and reference material for type erasure in C++.

- [cppreference — std::any](https://en.cppreference.com/w/cpp/utility/any) — Standard Library reference for `std::any`, `std::any_cast`, and `std::bad_any_cast`.
- [Arthur O'Dwyer — What is Type Erasure?](https://quuxplusone.github.io/blog/2019/03/18/what-is-type-erasure/) — Clear walkthrough of the pattern with diagrams and code.
- [Klaus Iglberger — Breaking Dependencies: Type Erasure (CppCon)](https://www.youtube.com/watch?v=jKt6A3wnDyI) — In-depth conference talk covering motivation, implementation, and comparisons with virtual dispatch.
- [Foonathan — Type Erasure](https://www.foonathan.net/2020/01/type-erasure/) — Practical guide to building your own type-erased wrapper step by step.
- [Jason Turner — C++ Weekly: std::any](https://www.youtube.com/watch?v=vLhr_y4YOIY) — Short video on `std::any` usage, pitfalls, and performance.
- [cppreference — std::function](https://en.cppreference.com/w/cpp/utility/functional/function) — Standard Library reference showing another major application of type erasure.
