# Optional Values — Links

Curated resources for understanding `std::optional` and its role in modern
C++ design.

- **cppreference — std::optional**
  <https://en.cppreference.com/w/cpp/utility/optional>
  Comprehensive API reference: constructors, observers, monadic operations
  (C++23), and deduction guides.

- **C++ Stories — Everything You Need to Know About std::optional**
  <https://www.cppstories.com/2018/05/using-optional/>
  Beginner-friendly walkthrough with practical examples of returning and
  consuming optionals.

- **Jason Turner — C++ Weekly Ep. 109: When to Use std::optional**
  <https://www.youtube.com/watch?v=UAAiwObNhSo>
  Short video discussing design guidelines and common pitfalls.

- **Sy Brand — Functional Exceptionless Error Handling with Optional and Expected**
  <https://blog.tartanllama.xyz/optional-expected/>
  Deep-dive into using optional and expected as monadic types for
  error handling without exceptions.

- **Bartek's Coding Blog — In-Place Construction for std::optional**
  <https://www.bfilipek.com/2018/07/in-place.html>
  Explains `std::in_place`, `emplace`, and how to avoid unnecessary copies
  when constructing optionals.

- **ISO C++ FAQ — When should I use optional?**
  <https://isocpp.org/wiki/faq/cpp17-library#optional>
  Official FAQ entry with rationale and idiomatic usage patterns.
